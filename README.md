# Smart Access Control System (智能门禁系统)

基于 STM32F103 的智能门禁系统，支持 RFID 刷卡、密码输入、Wi-Fi 远程控制、OLED 显示等多种功能。

## 功能特性

- **RFID刷卡开门**：支持 MIFARE RC522 模块，可识别用户卡和管理员卡
- **密码开门**：4x4矩阵键盘输入密码，支持用户密码和管理员密码
- **Wi-Fi远程控制**：通过 ESP-01S 模块连接巴法云平台，支持手机App远程开门/关门
- **舵机控制**：通过 PWM 控制 SG90 舵机实现门锁开关
- **OLED显示**：0.96寸 OLED 屏幕显示系统状态和操作提示
- **管理员功能**：
  - 修改用户密码/管理员密码
  - 添加/删除用户卡
  - 添加/删除管理员卡
- **数据存储**：使用片上 Flash 存储密码和卡片 ID，断电不丢失
  
  ## 硬件需求
  
  | 组件      | 型号/规格         | 说明              |
  | ------- | ------------- | --------------- |
  | 主控MCU   | STM32F103C8T6 | Cortex-M3       |
  | RFID模块  | RC522         | 13.56MHz        |
  | 舵机      | SG90          | 0°~180°         |
  | 显示屏     | 0.96寸 OLED    | I2C接口 (SSD1306) |
  | Wi-Fi模块 | ESP-01S       | AT指令控制          |
  | 键盘      | 4x4矩阵键盘       | GPIO扫描          |
  | 蜂鸣器     | 有源蜂鸣器         | 高电平触发           |
  | LED     | 红/绿双色         | 状态指示            |
  
  ## 引脚连接
  
  ### SPI接口 (RC522)
  
  | 引脚   | 功能       |
  | ---- | -------- |
  | PA4  | NSS (片选) |
  | PA5  | SCK (时钟) |
  | PA6  | MISO     |
  | PA7  | MOSI     |
  | PA11 | RST (复位) |
  
  ### 4x4矩阵键盘
  
  | 引脚              | 功能              |
  | --------------- | --------------- |
  | PA0~PA3         | 行输出 (ROW1~ROW4) |
  | PB0,PB1,PB3,PB4 | 列输入 (COL1~COL4) |
  
  ### 其他外设
  
  | 引脚      | 功能                  |
  | ------- | ------------------- |
  | PA8     | 舵机PWM (TIM1_CH1)    |
  | PA9     | USART1_TX (ESP-01S) |
  | PA10    | USART1_RX (ESP-01S) |
  | PA12    | 蜂鸣器                 |
  | PB8     | 红灯LED               |
  | PB9     | 绿灯LED               |
  | PB6,PB7 | I2C (OLED)          |

## Flash存储布局

| 地址         | 存储内容     |
| ---------- | -------- |
| 0x08007800 | 用户密码     |
| 0x08007C00 | 用户卡片ID   |
| 0x08008000 | 管理员卡片ID  |
| 0x08008400 | 新增用户卡片1  |
| 0x08008800 | 新增用户卡片2  |
| 0x08008C00 | 新增管理员卡片1 |
| 0x08009000 | 管理员密码    |
| 0x08009400 | 初始化标志位   |

## 使用说明

### 普通用户模式

1. 上电后显示 `Welcome` → `Please Open Door`
2. 输入6位用户密码，按确认键开门
3. 或直接刷卡开门（支持主卡和已添加的副卡）
4. 输入管理员密码可进入管理员模式
   
   ### 管理员模式
5. 输入管理员密码（默认000000）进入
6. **修改密码**：选择修改用户密码或管理员密码
7. **添加卡片**：刷未注册的新卡 → 选择添加为用户卡或管理员卡
8. **删除卡片**：从卡片列表中选择要删除的卡片并确认
   
   ### 远程控制
- 系统启动后自动连接 WiFi 并登录巴法云平台
- 通过手机App发送 `on` 命令可远程开门
- 发送 `off` 命令触发错误响应（红灯+蜂鸣器）
  
  ## 配置说明
  
  修改 `ESP.c` 中的配置参数：
  
  ```c
  char uid[]   = "997fb7889ff742939fe5860e41a201d1";  // 巴法云UID
  char topic[] = "led001";                            // 订阅主题
  char ssid[]  = "ACE";                               // WiFi名称
  char pswd[]  = "cr7204612";                         // WiFi密码
  ```

默认密码：

- 用户密码：`123456`

- 管理员密码：`000000`

默认卡片ID：

- 用户卡：`0xBAD3411D`

- 管理员卡：`0xC7F57B33`

## 工程说明

本工程使用 STM32CubeMX 生成 HAL 库代码，开发环境：

- IDE：Keil MDK 或 STM32CubeIDE

- 标准外设库：STM32F1xx_HAL_Driver

- 编译器：ARMCC 或 GCC

## 注意事项

1. RC522 与 STM32 通信使用软件模拟 SPI（模式0）

2. OLED 使用硬件 I2C1 接口（地址 0x78）

3. ESP-01S 使用 USART1（115200 baud），通过 DMA + 空闲中断接收数据

4. 舵机 PWM 频率为 50Hz（周期 20ms），脉宽 0.5ms~2.5ms 对应 0°~180°

5. Flash 写入前需要先擦除整页（每页 2KB）
